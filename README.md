# How does war affect inflation?

Research project for data analysis class. 
This project is an analysis of the relationship between war and inflation using global data.
It examines how inflation changes during periods of war.

# Important Files
- conflictdata.csv : data set 1
- inflationdata.xlsx : data set 2
- PSETTTTT444.qmd : raw Quarto document
- Problem Set 3.pdf : final pdf version for the project
